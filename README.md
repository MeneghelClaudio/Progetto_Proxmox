# 🛠️ PROXMOS AVENGERS

## 📌 Descrizione del progetto
**Proxmos Avengers** è una **web application basata su architettura a microservizi**, progettata per garantire scalabilità, modularità e facilità di manutenzione.

Il sistema è suddiviso in più microservizi indipendenti, ciascuno responsabile di una specifica funzionalità, comunicanti tra loro per offrire un'esperienza completa all'utente.

---

## 🏗️ Architettura

L'applicazione segue un'architettura a **microservizi**, con:

- Servizi backend indipendenti
- Frontend separato
- Database centralizzato
