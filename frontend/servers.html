Server Proxmox

Credenziali dei cluster / host Proxmox registrati

dns Aggiungi server Proxmox
Le credenziali vengono cifrate (Fernet AES-128) prima di essere salvate nel database.
Formato Proxmox: PVEAPIToken=utente@realm!tokenid=secret
Crea il token su Proxmox → Datacenter → Permissions → API Tokens.
info La password è necessaria solo per operazioni SSH sul cluster (join/leave/crea cluster). Le API Proxmox useranno sempre il token.
hub Crea Cluster
Raggruppa i server Proxmox registrati in un cluster logico. Il cluster viene gestito tramite il nodo primario.
